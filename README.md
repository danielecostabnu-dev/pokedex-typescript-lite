# Pokédex TypeScript Lite

Projeto desenvolvido em TypeScript para consulta e gerenciamento de Pokémon utilizando a PokeAPI.

## Funcionalidades

- Buscar Pokémon na PokeAPI
- Adicionar Pokémon ao catálogo
- Listar Pokémon cadastrados
- Buscar Pokémon por ID
- Remover Pokémon do catálogo
- Evitar registros duplicados

## Tecnologias utilizadas

- TypeScript
- Node.js
- PokeAPI
- Git
- GitHub

## Estrutura do Projeto

src/
├── controllers/
├── models/
├── services/
├── utils/
└── main.ts

## Como executar

Instalar dependências:

```bash
npm install
```

Executar projeto:

```bash
npx tsx src/main.ts
```
